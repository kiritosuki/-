# 接口文档

## 1. 病例管理

## 1.1 病例列表查询

### 1.1.1 基本信息

- 请求路径: /patients
- 请求方式: get
- 接口描述: 用于所有病例的列表查询 (分页查询 条件查询)

### 1.1.2 请求参数

- 参数格式: queryString

- 参数说明: 

 | 参数名称 | 是否必须 | 示例       | 备注                             |
 | :------- | -------- | ---------- | -------------------------------- |
  | name     | 否       | 张三/张/三 | 姓名                             |
  | gender   | 否       | 1          | 性别 1男 0女                     |
  | begin    | 否       | 2020-01-04 | 查询诊断日期(范围查询的起始日期) |
  | end      | 否       | 2022-01-22 | 查询诊断日期(范围查询的结束日期) |
  | page     | 是       | 1          | 页码 默认为1                     |
  | pageSize | 是       | 10         | 每页的数据行数 默认为10          |

- 请求参数样例: 

  /patients?name=张三&begin=2020-01-04&2022-01-22&page=1&pageSize=10
